# NoteSecured

NoteSecured is a Python-based secure note-taking application with a Tkinter GUI. It allows users to create and manage notes with basic security-focused design and local storage support.

## Features
- User authentication
- Secure note storage
- Tkinter desktop interface
- Local JSON-based data handling
- Client-server structure

## Project Files
- `client_gui.py` - GUI application
- `server.py` - server-side logic
- `auth.py` - authentication logic
- `security.py` - security utilities
- `storage.py` - data storage logic
- `test_app.py` - testing file

## How to Run
1. Make sure Python is installed.
2. Open the project folder in VS Code.
3. Run the main application file:
   ```bash
   python client_gui.py
